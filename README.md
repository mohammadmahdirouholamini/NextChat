# AI SaaS Platform (Next.js + OpenAI)

A modern AI SaaS platform built with **Next.js**, **TailwindCSS**, **Clerk Authentication**, and **OpenAI APIs**.

This project provides multiple AI tools in a single web application.

## Features

- 💬 AI Chat (GPT based)
- 🖼️ AI Image Generation
- 🔊 Text to Speech
- 🎤 Speech to Text
- 🔐 Authentication with Clerk
- 🎨 Modern UI with TailwindCSS
- ⚡ Fast Next.js App Router architecture

## Tech Stack

- Next.js
- TailwindCSS
- OpenAI API
- Clerk Authentication
- React
- TypeScript
